# jsonapi
Ruby gem for parsing/rendering [JSON API](http://jsonapi.org) documents.

## Installation
```ruby
# In Gemfile
gem 'jsonapi'
```
then
```
$ bundle
```
or manually via
```
$ gem install jsonapi
```
